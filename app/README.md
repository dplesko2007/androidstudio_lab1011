# Student Material Design App

## Название проекта
**Student_Material_Design** — приложение для отображения списка студентов с применением принципов Material Design. Каждый студент представлен в виде карточки с аватаркой, именем, возрастом и юмористическим описанием. Поддерживается раскрытие/сворачивание описания с плавной анимацией.

## Цель работы
Изучить и применить на практике принципы Material Design при разработке Android-приложений с использованием Jetpack Compose. В ходе лабораторной работы реализованы:
- кастомная цветовая схема (светлая и тёмная темы);
- типографика с пользовательскими шрифтами (Abril Fatface, Montserrat);
- различные формы компонентов (круглые аватарки, карточки с несимметричным скруглением);
- верхняя панель приложения (TopAppBar) с логотипом;
- анимация раскрытия/сворачивания карточек с помощью пружинной анимации (spring animation).

## Функциональность
- Отображение списка из 10 студентов.
- Каждый элемент списка содержит:
    - аватарку студента (круглая форма);
    - имя и возраст;
    - иконку раскрытия/сворачивания;
    - описание студента (появляется при нажатии на иконку).
- Плавная анимация при раскрытии/сворачивании карточки.
- Поддержка светлой и тёмной темы (автоматически подстраивается под настройки системы).
- Адаптивный дизайн с использованием отступов (dimens.xml).

## Технологии
- **Язык**: Kotlin
- **UI Toolkit**: Jetpack Compose
- **Дизайн‑система**: Material Design 3 (Material 3)
- **Шрифты**: Abril Fatface, Montserrat (Google Fonts)
- **Графика**: векторные и растровые иконки (Material Icons Extended)
- **Сборка**: Gradle (Kotlin DSL)
- **Среда разработки**: Android Studio
